Inspired by the famous minecraft server Wynncraft, a innovative lightweight framework for 2D RPG games.

> intent

A 2D game, with grinding possible, but also a relaxing atmosphere to hang out with friends! Started as a team project, ended up abondoned with a good foundation. Great place to learn and master your scripting skills.

> KEY://FEATURES

  • Database
  • Login/register screen
  • Hub main menu
  • Settings
  • It's own OST "Magical Forest" (credits to Veronika)
  • Multiple safe-zone cities
  • NPCs with a friendship system
  • Unique standalone quests & storylines
  • Balanced economy
  • PETS! We love pets, right??
  • New experience calculation method for levels
  • A nice user interface, featuring NPC dialogues, inventories, pet menus, settings, etc...
  • Items & gearsets with balanced stats
  • Save system & world settings synced to the server
  • Tile-based collision system
  • Uncenterable camera & cutscenes
  • Player owned plots and housing
  • Dueling
  • Trading
  • Farming
  • Fishing
  • Monsters with special abilities & rare bosses
  • Dungeons and world events
  • Day/night cycle
  • Different archetypes and classes
  • Wilderness
  • Detailed in-game and browser map
  • Discord RPC showing your current city and state, linking characters and much more
  • Server hosting made easy with one click! Controlable console
  • Admin mode for moderation without any gameplay benefits
  • Debug mode for admins
  • Easy map making and importing using Tiled
  • Surround sound engine
  • Shaders (WIP)
  • Different light sources (WIP)
  • And much much more!
